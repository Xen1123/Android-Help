# Prerequisites
- `adb` and `fastboot` Installed
- Unlocked Bootloader
- USB Debugging enabled
  - ## ADB/Fastboot On The 3 Most Popular Package Managers
    - Arch/Pacman -  `android-tools`
    - Debian/ATP - `adb` and `fastboot`
    - Fedora/DNF - `android-tools`
